KHU AI App Deployer API (1.0.0)

Download OpenAPI specification:

License: Internal research prototype

CPU/GPU VM 기반 AI 응용 등록·배포 프로토타입 API.

이 API는 AI App Spec 등록, Runtime/Target Profile 등록, 자원 readiness 점검, 배포 요청, 상태 조회, 로그 조회, 중지 요청을 제공한다.

1차년도 범위는 CPU VM, NVIDIA GPU VM, AI-Infra 연동 준비(Mock/Fixture)이며 Docker, Kubernetes, OCI Image, Container Registry 기반 배포는 포함하지 않는다.

System

서버 생존 상태와 준비 상태를 확인한다.

Process liveness check

API 서버 프로세스가 살아 있는지 확인한다. 저장소, Runtime Adapter, 외부 API 연결 상태는 검사하지 않는다.

Responses

Version 1 API prefix

/api/v1/healthz

Content type
application/json
{
  • "request_id": "req-test-001",
  • "status": "ok"
}

Profiles

Runtime Profile과 Target Profile을 등록하고 조회한다.

Register Runtime Profile

배포 실행 방식과 Runtime Adapter를 정의하는 Runtime Profile을 등록한다. CPU VM은 cpu/cpu_vm, GPU VM은 gpu/gpu_vm, AI-Infra 연동 준비는 aiinfra/etri_aiinfra 조합을 사용한다.

Request Body schema: application/json
required

등록할 Runtime Profile.

runtime_profile_id
required
string

Runtime Profile ID.

name
string

사람이 읽는 Runtime Profile 이름.

runtime_type
required
string
Enum: "mock" "cpu" "gpu" "aiinfra"

Runtime 종류. App Spec runtime.type과 호환되어야 한다.

accelerator
string
Enum: "none" "nvidia"

가속기 종류. GPU VM은 nvidia를 사용한다.

adapter_type
required
string
Enum: "mock" "cpu_vm" "gpu_vm" "etri_aiinfra"

사용할 Runtime Adapter. 서버 구현은 mock, cpu_vm, gpu_vm, etri_aiinfra를 등록한다.

operating_mode
required
string
Enum: "local_mock" "dry_run" "vm_process" "remote_api"

local_mock, VM 프로세스 실행, 원격 API 연동 모드.

object

외부 인터페이스 예제 호환용 readiness 점검 방식. 현재 구현은 저장만 수행한다.

  • "operating_mode": "vm_process"

Register Target Profile

배포 대상 VM, GPU, 저장소, Runtime 정보를 담은 Target Profile을 등록한다. 실제 Secret 값은 넣지 않고 credential_ref만 저장한다.

Request Body schema: application/json
required

등록할 Target Profile.

target_profile_id
required
string

Target Profile ID.

name
string

사람이 읽는 Target 이름.

csp
required
string
Enum: "aws" "azure" "gcp" "etri" "local" "mock"

Target 제공자. mock 외에는 구현 검증에서 vm.host가 필요하다.

region
string

CSP 리전. 선택 필드.

object

Target OS 정보. 선택 필드.

required
object

VM 또는 외부 Gateway 접근 정보. Secret 값은 저장하지 않고 credential_ref만 사용한다.

required
object

Target이 제공하는 Runtime 종류.

object

GPU VM Target 정보. gpu runtime일 때 gpu.count >= 1이 필요하다.

object

VM artifact/model/log 경로. CPU/GPU VM Adapter는 artifact_dir과 log_dir을 요구한다.

object

Target 네트워크 설정. 현재 구현은 저장만 수행한다.

Deployments

AI App 배포 요청, 상태 조회, 로그 조회, 중지 요청을 처리한다.

Record deployment inference metric

Store an inference or operation metric sample for a deployment. This is a year-1 placeholder path for latency, throughput, quality, and counters.

path Parameters
deployment_id
required
string

Deployment ID for the metric sample.

Request Body schema: application/json
required

Inference metric sample.

timestamp
string <date-time>

Metric sample time. If omitted, the server stores the current time.

latency_ms
number <double> >= 0

End-to-end inference latency in milliseconds.

throughput_rps
number <double> >= 0

Throughput in requests per second.

quality_score
number <double> [ 0 .. 1 ]

Optional normalized quality score.

request_count
integer >= 0

Number of observed requests in the sample window.

error_count
integer >= 0

Number of failed requests in the sample window.

object

Optional metric labels such as model, scenario, or dataset.
